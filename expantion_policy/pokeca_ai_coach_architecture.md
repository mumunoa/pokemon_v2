# ポケカAIコーチツール 完全アーキテクチャ設計（Next.jsベース）

## 前提

-   フロントエンド：Next.js（既にUIとカード操作を実装中）
-   本設計では **既存UIにバックエンド / AI / データ基盤を追加する構造**
    にする
-   目的：3ヶ月で課金可能なMVPを作る

------------------------------------------------------------------------

# 全体アーキテクチャ

    Next.js Frontend
    │
    ├─ UI Layer
    │   ├ デッキ画面
    │   ├ 一人回し画面(1デッキ)
    │   ├ 一人回し画面(2デッキ)
    │   ├ AI分析画面
    │   └ メタ環境画面
    │
    ├─ API Layer (Next.js API Routes / Backend)
    │   ├ Deck Service
    │   ├ Practice Engine
    │   ├ AI Coach Service
    │   └ Analytics Service
    │
    ├─ Data Layer
    │   ├ PostgreSQL
    │   ├ Redis
    │   └ Game Logs
    │
    └─ AI Engine (Python)
        ├ Monte Carlo Simulation
        ├ Strategy Evaluation
        └ Training Data
