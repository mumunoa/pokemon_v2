# サンドボックスUI・インタラクションルール

本文書は、TCGのサンドボックス（一人回し）機能における各ゾーンのUIおよび状態遷移の仕様を定義する。

## 1. 共通ルール・操作
- **ドラッグ＆ドロップ（D&D）**:
  - 幅768px以下のモバイルデバイス（iPhone/iPad等）でも、画面全体がスクロールせずにD&Dが正常に動作すること。
  - カードが縦向き・横向き・裏向きに関わらずD&D可能であること。ドロップ時には、状態異常による回転を除き、デフォルトの正面向きにリセットされる。
- **カードの表裏の定義**:
  - 「裏向き」とは「カードの背面（共通のカード裏面画像）」が表示されている状態を指し、画像が180度回転することではない。
- **相手視点切替**:
  - メイン画面の右中央にあるボタンは「相手視点に切替」とする。
- **コイントス**:
  - 画面上部のコイントスはボタンではなく「丸いコインマーク」として配置する。
  - タップorクリックするとコインが回転するアニメーションが再生され、表面か裏面の画像で結果が表示されること。
- **エネルギーの表示**:
  - ポケモンにエネルギーカードが重なった（つけられた）場合、ポケモンの左下にエネルギーのシンボルマークを表示する。
  - マークはついた順番に左から並ぶ（例: 草、雷、雷）。
  - 各エネルギーマークは `design_rules/` 以下の画像（grass.png, fire.png, water.png, lightning.png, psychic.png, fighting.png, darkness.png, metal.png, dragon.png, colorless.png）を参照して生成する。

## 2. ゾーン別詳細仕様

### サイド (Prizes)
- **配置**: 裏向きかつ横向き。縦に6枚少しずつ重なるように配置。
- **タップ時のアクション**:
  - 背景を暗くし、ポップアップを表示。カードを上3枚・下3枚に大きく裏面表示する。
  - ポップアップ外をタップで閉じる。
- **ポップアップ内のボタン**:
  1. **カードを表にする / カードを裏にする**: トグル式。現在の状態に応じてラベルを変え、一括で表裏を切り替える。
  2. **シャッフルする**: カードの並び順をランダムに入れ替え、ポップアップを閉じる。
  3. **1枚取る / 2枚取る / 3枚取る**: 選択した枚数分、サイドからカードを手札に表向きで移動し、ポップアップを閉じる。サイドが0枚になった場合は「勝利！」画面を表示。

### 山札 (Deck)
- **盤面UI**: 山札の上に「1枚ドロー」「シャッフル（design_rules/shuffle.svgアイコン）」ボタンを配置。シャッフル時にアニメーションを実行。
- **タップ時のアクション（ポップアップ）**:
  - カードが横一列に重なって並び、10枚程度が見える状態。スワイプで横スクロール可能。
- **ポップアップ内のボタン**:
  1. **1枚ドロー**: 1番上のカードを手札に追加し閉じる。
  2. **上5枚見る / 上7枚見る**: 上から指定枚数を表向きにする。うち1枚を上か下に半分以上ドラッグするとポップアップが閉じ、そのままフィールドへD&D可能。途中でドロップ（無効な位置）した場合は元の「○枚見る」状態に戻る。
  3. **山札を見る**: 全てのカードを表向きにする。ドラッグの仕様は上記と同様。
  4. **シャッフル**: アニメーション後、ランダムに並べ替えてポップアップ状態に戻る。
  5. **裏向きにする**: 表向きのカードをすべて裏向きに戻す（順番維持）。

### トラッシュ (Trash)
- **配置**: 最新のカードを一番上にして表向きで配置。
- **タップ時のアクション（ポップアップ）**:
  - 「山札を見る」と同様に、全カードが表向きで横滑りするスライダーUIを表示。
  - 1枚を上下にドラッグすることでフィールドに取り出せる。

### ベンチ (Bench)
- **配置制限**: 「ポケモン」と「ポケモンのどうぐ」のみ配置可能。どうぐはポケモンの下に重なる（Zインデックス調整）。
- **エネルギーの付与**: エネルギーをポケモンにドロップすると、左下にマーク化して即座に付与される。
- **タップ時のアクション（ポップアップ）**:
  - カード名、ダメージ操作ボタン、特性ボタンを表示（向き変更ボタンはなし）。
  - **ダメージ操作**: `+10`, `+50`, `+100`, `-10`, `-50`, `-100`, `リセット(0)`。ポップアップ内の数字が即時反映され、最大値は「1000」とする。ポップアップを閉じた際にも盤面に反映されること。
  - **特性**: APIから取得した特性名（または「特性を使用する」）ボタン。タップで色を変えて選択状態化。このまま閉じるとカード中央に「USEDマーカー」を付与。再度タップでトグル解除。

### バトル場 (Active)
- **基本仕様**: ベンチの全機能（エネルギー付与、即時ダメージ反映、特性）に加え、「特殊状態」の管理を含む。
- **タップ時のアクション（ポップアップ）**:
  - ダメージ、特性の横に、特殊状態ボタンを追加。タップすると即時に盤面のカードにマーカーが配置される。
- **特殊状態の仕様**:
  - **どく**: トグル選択。ポケモンのHPバー（またはカード上）に「どくマーカー」を即時配置。
  - **やけど**: トグル選択。ポケモンのカード上に「やけどマーカー」を即時配置。
  - **まひ**: トグル選択。カード上に「まひマーカー」を配置し、カードを**90度回転**。
  - **ねむり**: トグル選択。カード上に「ねむりマーカー」を配置し、カードを**90度回転**。
  - **こんらん**: トグル選択。カード上に「こんらんマーカー」を配置し、カードを**180度回転**。
  - ※「ねむり」「まひ」「こんらん」は排他処理（重複不可）。
